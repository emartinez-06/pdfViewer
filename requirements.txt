# Docker Requirements for PDF Viewer Application

## System Requirements
- Docker Engine 20.10+
- Docker Compose 2.0+
- Minimum 2GB RAM available for containers
- Minimum 1GB disk space

## Node.js Dependencies (handled automatically by Docker)

### Backend Dependencies:
- Node.js 18 (Alpine Linux base)
- express ^4.18.2
- cors ^2.8.5
- dotenv ^16.3.1
- helmet ^7.0.0
- morgan ^1.10.0

### Frontend Dependencies:
- Node.js 18 (Alpine Linux base)
- react ^18.2.0
- react-dom ^18.2.0
- react-scripts 5.0.1
- react-router-dom ^6.15.0
- axios ^1.5.0

### Web Server:
- Nginx (Alpine Linux base) for serving React app

## Deployment Instructions

1. Install Docker and Docker Compose on your system
2. Clone the repository
3. Navigate to the project root directory
4. Build and start the services:
   ```bash
   docker-compose up --build
   ```

## Service Endpoints
- Frontend: http://localhost:80
- Backend API: http://localhost:3001
- Health Check: http://localhost:3001/health

## Environment Variables (Optional)
Create a .env file in the backend directory:
```
NODE_ENV=production
PORT=3001
```

## Storage
- Backend uploads are persisted in ./backend/uploads volume
- Optional PostgreSQL database volume (commented out by default)

## Commands
- Start services: `docker-compose up -d`
- Stop services: `docker-compose down`
- View logs: `docker-compose logs -f`
- Rebuild: `docker-compose up --build`
